# earthquake-ai-app

Project Description (README)
Earthquake AI

Earthquake AI is a machine learning project developed to predict and estimate the severity of earthquakes and the potential damage that may occur after an earthquake event.

This project uses three main seismic input features:

Depth – the depth of the earthquake
Magnitude – the strength of the earthquake
Intensity – the level of shaking or impact felt

By analyzing these three factors, the system can estimate:

Earthquake severity level
Possible damage level
Impact classification for decision support

The goal of this project is to provide an intelligent tool that can help in earthquake impact assessment, risk awareness, and disaster preparedness.
