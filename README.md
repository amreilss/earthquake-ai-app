## Earthquake AI

**Earthquake AI** is an intelligent prediction application designed to **analyze and estimate the severity of earthquakes** as well as the **possible damage that may occur** based on seismic data.

The prediction model is built using three key earthquake-related parameters:

- **Depth**
- **Magnitude**
- **Intensity**

By processing these input values, the system can predict:

- **Earthquake severity level**
- **Potential destruction or damage level**
- **Overall impact in affected areas**

To improve accessibility and usability, the project was developed as an **application** that enables users to interact with earthquake prediction results in a more convenient and practical way.

The application is integrated with the **map of Thailand** to provide geographic visualization of:

- **Earthquake-prone areas**
- **High-risk zones**
- **Potentially affected locations**

In addition, the system includes an **alert notification feature** that can notify users who are located near **dangerous or high-risk areas**, allowing them to **prepare and respond to possible earthquake events in a timely manner**.

The purpose of this project is to support:

- **Earthquake risk assessment**
- **Disaster impact analysis**
- **Preparedness and emergency response planning**

**Earthquake AI** aims to apply artificial intelligence in disaster monitoring and public safety to help reduce risk, improve awareness, and support faster decision-making.
